# robustify.js

robustify.js is javascript that attempts to fight [link rot](https://en.wikipedia.org/wiki/Link_rot) or content drift with an implementation of the [Memento Robust Links - Link Decoration](http://robustlinks.mementoweb.org/spec/) specification.

See the example.html file for examples how to use robustify.js.

robustify.js should work on any modern browser or IE 8 or better.
